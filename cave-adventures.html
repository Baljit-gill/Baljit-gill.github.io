Cave Adventures

A puzzle platformer built in Python using Pygame.

Overview

Cave Adventures is a 2D Pygame platformer where the player explores dark caves, collects keys, and unlocks doors to progress through levels. The game includes enemies, hazards, and a growing difficulty curve.

Features

  • Key–door mechanic with multiple levels
  • Player movement, jumping, collision physics
  • Collectibles and treasure items
  • Enemies that chase or patrol
  • Sound effects & music integration
  • Sprite animations

Technologies Used

Python, Pygame, Sprite Sheets, Game Physics
