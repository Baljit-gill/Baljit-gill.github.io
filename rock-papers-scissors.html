Rock, Paper, Scissors Game

A simple C program where the player battles the computer in 5 rounds.

Overview

This Rock–Paper–Scissors game was coded in C and allows the player to compete against the computer over 5 rounds. The computer's move is generated using randomisation.

How It Works

  • Player chooses Rock, Paper, or Scissors
  • Computer randomly selects a choice
  • Game tracks wins, losses, and draws
  • Winner announced at the end of 5 rounds

Technologies Used

C (rand function), Loops, Conditionals, User Input
